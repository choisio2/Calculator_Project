# Calculator Project

## 1. 프로젝트 개요 
- **프로젝트명**: Calculator App  
- **개발 환경**: Android Studio, Kotlin  
- **목표**: 기본적인 사칙연산(+, -, ×, ÷)이 가능한 계산기 애플리케이션 개발  

---

## 2. 주요 기능
- 숫자 입력 (0~9)  
- 사칙연산 버튼 (+, -, ×, ÷)  
- CE(Clear Entry): 이전 한 글자만 지우기 
- AC(All Clear): 전체 초기화
- "=": 결과 출력  
- 연속 연산 지원 (예: `2 + 3 × 4`)  
- Unary minus 지원
